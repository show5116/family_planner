# 패밀리플래너 v1.1.3 업데이트

> 출시일: 2026-07-07

## 주요 변경사항

### ✨ 새로운 기능
- **고정지출 소급 등록 및 할부**: 고정지출을 등록할 때 시작월을 지정해 이전 달로 소급 등록할 수 있고, 반복 종료 개월수를 지정해 할부처럼 일정 기간만 반복되도록 설정할 수 있습니다.
- **가족 초대 공유 기능**: 이메일 초대 대신 딥링크로 초대 링크를 공유해 가족을 더 쉽게 초대할 수 있습니다.

### 🐛 버그 수정
- **월간 캘린더**: 캘린더 그리드에서 세로로 드래그할 때 페이지 전체가 스크롤되던 문제 수정
- **주간 타임테이블**: 헤더와 그리드 사이 경계선이 어긋나 보이던 문제 수정
- **가계부**: 지출 내역을 수정하거나 등록해도 화면에 바로 반영되지 않던 문제 수정
- **적금 알림**: 알림을 눌렀을 때 엉뚱한 화면으로 이동하던 문제 수정
- **공지사항**: 웹 에디터로 작성한 서식이 깨지던 문제, 미리보기에서 HTML 태그가 그대로 보이던 문제 수정
- **장보기**: 장보기 완료 시 소비처 목록이 표시되지 않던 문제 수정

### 🔧 개선사항
- **월간 캘린더**: 날짜 상세 정보 접근성 개선
- **일정**: 종일 일정이 00시에 표시되지 않도록 개선
- **가족 초대 링크**: 소개 사이트 경로와 충돌하지 않도록 딥링크 경로 구조 변경
