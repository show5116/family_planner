# 패밀리플래너 v1.1.2 업데이트

> 출시일: 2026-06-30

## 주요 변경사항

### 🐛 버그 수정
- **Apple로 로그인**: 로그인 후 이름/이메일을 다시 입력하라고 요구하던 화면을 제거. Apple이 이미 제공한 정보를 그대로 사용하도록 개선 (App Store 심사 가이드라인 대응)
- **알림 카테고리**: 냉장고 알림이 일부 알림 화면에서 아이콘/색상 없이 표시되던 문제 수정
- **대시보드 홈 위젯**: 위젯이 제대로 적용되지 않던 현상 수정
- **자산 기록**: 자산 기록 입력 시트가 화면 안전영역을 침범하던 문제 수정
- **고정 메모**: 고정한 메모가 갱신되지 않던 현상 수정
- **블루투스 키보드**: 블루투스 키보드 사용 중 입력 화면이 초기화되던 문제 수정
- **장보기**: 장보기 완료 버튼이 화면 하단 안전영역에 가려지던 문제 수정

### 🔧 개선사항
- **소셜 로그인 버튼 디자인**: Apple/Google 로그인 버튼을 각 사 공식 브랜드 가이드라인에 맞춰 새로 디자인
