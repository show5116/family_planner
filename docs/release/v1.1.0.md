# 패밀리플래너 v1.1.0 업데이트

> 출시일: 2026-06-22

## 주요 변경사항

### ✨ 새로운 기능

- **기념일 관리**: 가족의 생일, 결혼기념일 등 중요한 날을 등록하고 대시보드에서 한눈에 확인할 수 있습니다
- **기념일 대시보드 위젯**: 다가오는 기념일을 홈 화면에서 바로 확인할 수 있는 위젯이 추가되었습니다
- **Apple 로그인**: iPhone/iPad 사용자도 Apple 계정으로 간편하게 로그인할 수 있습니다
- **광고 시스템**: 앱 내 광고가 적용되었습니다. 신규 회원가입 후 2주간은 광고가 표시되지 않습니다
- **Google Analytics 연동**: 앱 사용 통계를 수집하여 더 나은 서비스를 제공합니다 (프로덕션 환경에서만 작동)

### 🐛 버그 수정

- 할일 목록에서 항목 순서를 변경할 때 발생하던 오류를 수정했습니다
- iOS에서 일부 화면이 정상적으로 동작하지 않던 문제를 수정했습니다
- iOS 맞춤 튜토리얼 화면의 오류를 수정했습니다

### 🔧 개선사항

- 메모의 체크리스트 항목이 변경 즉시 자동 저장되도록 개선되었습니다
- 튜토리얼 화면 구성이 더 명확하게 분리되어 안내가 개선되었습니다
- 홈 화면 위젯 설정이 대시보드 중심으로 개편되었습니다
- 기념일 조회 시 날짜 필터가 추가되어 원하는 기간의 기념일만 볼 수 있습니다

### 🔒 설정 및 플랫폼

- iOS 배포 환경 설정 완료 (App Store 출시 대응)
- Apple 로그인 크로스 플랫폼 지원 (iOS/Android 통합)
- 날씨 정보를 iOS에서도 정상적으로 불러올 수 있도록 수정
