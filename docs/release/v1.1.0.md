# 패밀리플래너 v1.1.0 업데이트

> 출시일: 2026-06-25

## 주요 변경사항

### ✨ 새로운 기능

- **기념일 관리**: 가족의 생일, 결혼기념일 등 중요한 날을 등록하고 대시보드에서 한눈에 확인할 수 있습니다
- **기념일 대시보드 위젯**: 다가오는 기념일을 홈 화면에서 바로 확인할 수 있는 위젯이 추가되었습니다
- **Apple 로그인**: iPhone/iPad 사용자도 Apple 계정으로 간편하게 로그인할 수 있습니다
- **일정 캘린더 다중 뷰**: 일별·주별·월별·연별 보기를 지원하며, 종일 일정 및 빠른 일정 생성 기능이 추가되었습니다
- **가계부 조회 기능**: 기간별 지출 내역 조회 UI와 고정 지출 리스트 및 평균값 조회 기능이 추가되었습니다
- **운영자 권한 설정**: 그룹 내 운영자 권한을 설정할 수 있는 기능이 추가되었습니다
- **가계 자금 이동**: 기존 이월 기능이 계좌 간 자금 이동 기능으로 강화되었습니다
- **소셜 회원가입 정보 입력**: 소셜 로그인 신규 가입 시 추가 정보를 입력하는 단계가 추가되었습니다
- **광고 시스템**: 앱 내 광고가 적용되었습니다. 신규 회원가입 후 2주간은 광고가 표시되지 않습니다

### 🐛 버그 수정

- 일정 화면의 UX 및 캘린더 내 텍스트 표시 오류를 수정했습니다
- 비로그인 상태에서 일부 API가 인증 없이 호출되지 않던 문제를 수정했습니다
- 가계부 고정 항목의 예상 금액이 당일 데이터를 잘못 포함하던 현상을 수정했습니다
- 가계 자산 이동 기능의 버그 및 UX 오류를 수정했습니다
- 할일 목록에서 항목 순서를 변경할 때 발생하던 오류를 수정했습니다
- iOS에서 일부 화면이 정상적으로 동작하지 않던 문제를 수정했습니다

### 🔧 개선사항

- 드롭다운·바텀시트 닫힌 후 이전 포커스를 복구하지 않도록 변경하여 UX가 개선되었습니다
- 앱 최초 설치 시 라이트 모드로 시작하도록 변경되었습니다
- 홈 화면 위젯 설정 UI가 개선되고 대시보드 기념일 위젯이 강화되었습니다
- 기념일 조회 시 날짜 필터가 추가되어 원하는 기간의 기념일만 볼 수 있습니다
- 메모의 체크리스트 항목이 변경 즉시 자동 저장되도록 개선되었습니다
- 테스트 전용 알림 버튼이 일반 사용자에게 노출되지 않도록 처리했습니다

### 🔒 설정 및 플랫폼

- iOS 배포 환경 설정 완료 (App Store 출시 대응)
- Apple 로그인 크로스 플랫폼 지원 (iOS/Android 통합)
- 날씨 정보를 iOS에서도 정상적으로 불러올 수 있도록 수정
