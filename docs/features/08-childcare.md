- ⬜ 자녀별 포인트 카드
- ⬜ 포인트 사용/적립 폼
- ⬜ 포인트 표 관리 화면
- ⬜ 포인트 규칙 관리 화면
- ⬜ 포인트 히스토리 화면

## 데이터 모델
- ⬜ 자녀 모델 (Child)
- ⬜ 포인트 계정 모델 (PointAccount)
- ⬜ 포인트 항목 모델 (PointItem)
- ⬜ 포인트 규칙 모델 (PointRule)

- ⬜ 매달 정해진 금액 포인트 자동 지급
- ⬜ 부모의 포인트 지급액 설정
- ⬜ 포인트 적금 기능
- ⬜ 적금 이자 계산 및 지급
- ⬜ 포인트 사용 항목 등록/편집
- ⬜ 포인트 규칙 등록/편집
- ⬜ 규칙 위반 시 포인트 차감
- ⬜ 포인트 적립/사용 내역 조회
- ⬜ 날짜별 필터링

## API 연동

- ⬜ 포인트 계정 조회 API
- ⬜ 포인트 적립/사용 API
- ⬜ 포인트 항목 관리 API
- ⬜ 포인트 규칙 관리 API
