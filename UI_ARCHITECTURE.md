## 네비게이션 구조

### 전체 구조 방식
**Drawer + Bottom Navigation 혼합 방식**

#### Bottom Navigation (주요 5개 메뉴)
하단 네비게이션 바에 가장 자주 사용되는 메뉴 배치:

```
┌─────────────────────────────────────┐
│                                     │
│         Content Area                │
│                                     │
├─────────────────────────────────────┤
│ 🏠    📊    📅    ✅    👤          │
│ 홈   자산   일정   할일   더보기     │
└─────────────────────────────────────┘
```

1. **홈** (Home) - 메인 대시보드
2. **자산** (Assets) - 자산관리 + 투자지표
3. **일정** (Calendar) - 일정 관리
4. **할일** (Tasks) - ToDoList
5. **더보기** (More) - 나머지 메뉴 + 설정

#### Drawer Menu (전체 메뉴)
"더보기" 탭 또는 좌측 Drawer에서 접근:

```
┌──────────────────────┐
│  Family Planner      │
│  [프로필 영역]        │
├──────────────────────┤
│ 🏠 홈                │
│ 💰 자산관리          │
│ 📈 투자지표          │
│ 💳 가계관리          │
│ 📅 일정관리          │
│ ✅ ToDoList         │
│ 👶 육아포인트        │
│ 📝 메모              │
│ 🎮 미니게임          │
├──────────────────────┤
│ ⚙️  설정             │
│ 👥 가족관리          │
│ 🔔 알림              │
│ ❓ 도움말            │
│ 🚪 로그아웃          │
└──────────────────────┘
```

## 정보 구조

### 메인 대시보드 구성

메인 화면은 **위젯 기반 커스터마이징 대시보드**:

```
┌─────────────────────────────────┐
│ 상단바 (프로필, 알림, 설정)      │
├─────────────────────────────────┤
│                                 │
│  [위젯 1: 오늘의 일정]          │
│                                 │
├─────────────────────────────────┤
│                                 │
│  [위젯 2: 투자 지표 요약]       │
│                                 │
├─────────────────────────────────┤
│                                 │
│  [위젯 3: 할일 요약]            │
│                                 │
├─────────────────────────────────┤
│                                 │
│  [위젯 4: 이번 달 지출]         │
│                                 │
├─────────────────────────────────┤
│  + 위젯 추가                    │
└─────────────────────────────────┘
```

#### 사용 가능한 위젯 종류
1. **오늘의 일정** - 당일 일정 3개 표시
2. **투자 지표 요약** - 코스피, 나스닥, 환율 (간단 차트)
3. **할일 요약** - 진행 중인 할일 5개
4. **자산 요약** - 총 자산, 수익률
5. **이번 달 지출** - 카테고리별 지출 (원형 차트)
6. **육아 포인트** - 자녀별 포인트 현황
7. **빠른 메모** - 최근 메모 3개
8. **가족 현황** - 가족 구성원 활동 요약

## 반응형 디자인

### 브레이크포인트
```dart
// Mobile: < 600px
// Tablet: 600px ~ 1024px
// Desktop: > 1024px

class Breakpoints {
  static const double mobile = 600;
  static const double tablet = 1024;
}
```

### 레이아웃 전략
```dart
// 모바일: Single Column
// 태블릿: Two Column (일부 화면)
// 데스크톱: Side Navigation + Content

LayoutBuilder(
  builder: (context, constraints) {
    if (constraints.maxWidth < 600) {
      return MobileLayout();
    } else if (constraints.maxWidth < 1024) {
      return TabletLayout();
    } else {
      return DesktopLayout();
    }
  },
)
```

## 접근성 (Accessibility)

### 주요 고려사항
1. **색상 대비**: WCAG AA 기준 이상 (4.5:1)

// Semantics 예시
